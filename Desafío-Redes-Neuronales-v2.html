Codificacin las variables

Utilizamos la funcin dummy para crear variables dummy para realizar onehot encoding Las variables dummy son una tcnica comn en estadstica y aprendizaje automtico para representar datos categricos o nominales de manera binaria. El proceso de one-hot encoding implica convertir cada categora en una columna separada, donde se utiliza un valor # binario (0 o 1) para indicar si pertenece o no a esa categora.

dummy <- dummyVars(" ~ .", data=heart)
newdata <- data.frame(predict(dummy, newdata = heart)) 
newdata$ChestPainType <- NULL
newdata$RestingECG <- NULL
newdata$ST_Slope <- NULL
data <- newdata
