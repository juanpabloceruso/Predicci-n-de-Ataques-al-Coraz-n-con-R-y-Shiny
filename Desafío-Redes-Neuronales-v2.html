Codificación las variables

Utilizamos la función dummy para crear variables dummy para realizar onehot encoding Las variables dummy son una técnica común en estadística y aprendizaje automático para representar datos categóricos o nominales de manera binaria. El proceso de one-hot encoding implica convertir cada categoría en una columna separada, donde se utiliza un valor # binario (0 o 1) para indicar si pertenece o no a esa categoría.

dummy <- dummyVars(" ~ .", data=heart)
newdata <- data.frame(predict(dummy, newdata = heart)) 
newdata$ChestPainType <- NULL
newdata$RestingECG <- NULL
newdata$ST_Slope <- NULL
data <- newdata
